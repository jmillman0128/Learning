Basics 1

In this document, you will find references to various types of things you can add to an HTML file.

HTML is a mark up language meant for storing content for a website. All manner of things like text, links, videos, and beyond can be added here.

Information can be added to an HTML file using objects, which allow for basic formatting for text, and permit non-text to manifest.

Objects can be added by encasing the appropriate element in a tag (<>) like so:<x><\x>

You can italicize using "em" (visually and for screen readers) or "i" (visually)

You can bold using "strong" (visually and for screen readers) or "b" (visually)

You can underline using "u" (visually)

You can group items together using sections by applying "sections" a line above and below your content.

You can break a line by using "br"

Comments are not visible to visitors, but will provide extra detail to other developers and are created using "!--", only use a ! at the start.

You can add a link using "a", this one leads to the next Basics HTML file
You must include the link or destination within the first tag using "href =" followed by the link address

You can add an image using "img", this one is a placeholder
You must include the link to the image within the first tag using "src =" followed by the link address
You can also specify attributes by listing the width and height in pixels, as well as add a caption for a screen reader using alt.
